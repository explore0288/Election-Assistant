# Election Assistant AI

An AI-powered chatbot designed to answer election-related questions using the Gemini API.

## How to Run hohukkj
1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. You will be prompted to enter a **Gemini API Key**. 
   - You can get a free key at [Google AI Studio](https://aistudio.google.com/).

## Tech Stack
- HTML5 / CSS3
- JavaScript (Vanilla)
- Google Gemini 1.5 Flash API
